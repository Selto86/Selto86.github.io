Loup (Projet GameJam)

Moteur utilisé : Godot
Type de jeu : Arcade
Projet solo en Game Jam (une semaine)

Loup est un jeu que j'ai développé durant la Jampley Game Jam sur itch.io sur une semaine de mars 2025. Le jeu est un jeu d'arcade inspiré du jeu du Loup ; Le joueur incarne un loup qui a pour but de manger tous les moutons, qui se déplacent aléatoirement sur la scène. Les moutons possèdent cependant une cabane au centre du tableau permettant de se cacher, et qui va rétrécir à chaque mouton tué ! Les moutons vous évitent quand vous vous approchez, alors prenez garde !
Ce projet m'a permis d'étoffer mes capacités en Game Design grâce à des rajouts de modes de difficulté, d'effets sonores et visuels, d'IA ennemie adaptative, et de menus.
Ce projet est jouable sur navigateur !
